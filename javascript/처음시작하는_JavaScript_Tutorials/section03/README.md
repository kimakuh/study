## 10 - JavaScript 객체

### 객체
- 프로그램에서 매게체 하나에 대한 값과 기능을 관리하는 단위
- 자바스크립트는 객체를 다음 두 가지 방법으로 생성할 수 있음
 - 리터럴을 이용한 방법
 - 생성자 함수를 이용한 방법


### 리터럴를 이용한 객체 생성
- 자바스크립트에서는 객체를 {}를 이용해 정의
- 객체는 정의됨과 동시에 생성



<pre>
    <script>
        var obj1 = {
            // 객체를 {} 안에 만들고 obj1 안에 넣는다
            a1 : 100,
            "a2" : 11.11,
            a3 :  true,
            a4 : "안녕하세요",
            a5 : [10, 20, 30],
            a6 : { // 객체 안에 객체 생성가능
                
                b1 : 100,
                b2 : 200,

            },
            a7 : function(){ // 객체 안에 함수도 생성가능
                document.write("a7 함수 호출<br/>");
            }
        }
    </script>
</pre>
