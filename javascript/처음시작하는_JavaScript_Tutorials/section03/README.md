## 10 - JavaScript 객체

### 객체
- 프로그램에서 매게체 하나에 대한 값과 기능을 관리하는 단위
- 자바스크립트는 객체를 다음 두 가지 방법으로 생성할 수 있음
 - 리터럴을 이용한 방법
 - 생성자 함수를 이용한 방법


### 리터럴를 이용한 객체 생성
- 자바스크립트에서는 객체를 {}를 이용해 정의
- 객체는 정의됨과 동시에 생성


<pre>

		function f1(){
			document.write("f1함수 호출");

		// 함수를 호출해야 내부에 있는 코드가 출력
		
		f1();
	</script>
