## JavaScript 작성 위치

### inline 방식
- HTML 태그 내에 JavaScript 코드를 작성
- 보통 태그에 대한 이벤트를 처리하기 위해 사용
<pre><button onclick="alert('버튼클릭')">버튼</button></pre>

### internal 방식
- HTML 문서 내부에 JavaScript 코드를 작성하며 script 태그 사이에 작성
- 함수를 정의하여 사용하거나 페이지가 나타날 때 자동으로 실행되는 코드가 필요할 때 사용
